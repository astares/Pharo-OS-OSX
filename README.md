# Pharo-OS-OSX

## Introduction
This is a Pharo package written by Torsten Bergmann to get better support for OS X Apple systems within [Pharo](http://www.pharo.org)

```Smalltalk
Metacello new 
	repository: 'github://astares/Pharo-OS-OSX/src';
	baseline: 'OSOSX' ;
	load
```
